# Криптовалюты и безопасность

## Простыми словами

Криптовалюта — это цифровые деньги без банка. Они хранятся в «кошельках», доступ к которым даёт приватный ключ. Потеряли ключ — потеряли деньги навсегда. Нет «забыли пароль?», нет «службы поддержки».

## Как работает криптовалюта?

### Без банка:

**Традиционные деньги:**
Вы → Банк → Получатель
Банк подтверждает: «Да, у вас есть деньги»

**Криптовалюта:**
Вы → Блокчейн → Получатель
Блокчейн подтверждает: «Да, перевод подписан вашим ключом»

### Приватный и публичный ключ:

**Публичный ключ** — как номер счёта, можно всем говорить
**Приватный ключ** — как пароль от счёта, никому не говорить

## Где хранить криптовалюту?

### Типы кошельков:

| Тип | Безопасность | Удобство | Для кого |
|-----|--------------|----------|----------|
| Биржевой кошелёк | Низкая | Очень высокое | Трейдинг |
| Мобильный кошелёк | Средняя | Высокое | Повседневные платежи |
| Аппаратный кошелёк | Очень высокая | Среднее | Долгосрочное хранение |
| Бумажный кошелёк | Очень высокая | Низкое | Холодное хранение |

### Биржевые кошельки (небезопасно!)

**Плюсы:**
- Легко покупать/продавать
- Не нужно ничего настраивать
- Забыли пароль — восстановят

**Минусы:**
- ✅ Биржа владеет ключами, а не вы
- ✅ Биржу могут взломать
- ✅ Биржу могут закрыть
- ✅ Нет анонимности

**Истории:**
- Mt. Gox — украли 450 миллионов $
- FTX — обанкротились, миллиарды $ заморожены

### Мобильные кошельки

**Популярные:**
- Trust Wallet
- MetaMask
- Rainbow

**Плюсы:**
- Удобно для небольших сумм
- Приватный ключ на вашем телефоне

**Минусы:**
- ❌ Если взломают телефон — потеряете
- ❌ Если потеряете телефон + seed-фразу — потеряете

### Аппаратные кошельки (самые безопасные)

**Популярные:**
- Ledger
- Trezor

**Плюсы:**
- ✅ Приватный ключ хранится офлайн
- ✅ Не украдут через интернет
- ✅ Подтверждение на устройстве

**Минусы:**
- ❌ Стоит 50-200$
- ❌ Нужно не потерять
- ❌ Нужно не забыть seed-фразу

### Seed-фраза (самое важное!)

Это 12-24 слова, которые восстанавливают кошелёк.

**Правила:**
1. Запишите на бумаге
2. Храните в надёжном месте (сейф, банковская ячейка)
3. НЕ храните в телефоне/компьютере
4. НЕ фотографируйте
5. НЕ говорите никому

**Пример seed-фразы:**
```
witch collapse practice feed shame open despair creek road again ice least
```

## Типичные атаки

### 1. Фишинг

**Схема:**
- Получаете email «войдите в кошелёк»
- Ссылка на поддельный сайт
- Вводите seed-фразу
- Мошенник крадёт деньги

**Защита:**
- Всегда проверяйте URL
- Никогда не вводите seed-фразу онлайн
- Используйте аппаратный кошелёк

### 2. Вредоносное ПО

**Схема:**
- Скачиваете «кошелёк» с неофициального сайта
- Программа ворует приватный ключ
- Деньги уходят мошеннику

**Защита:**
- Скачивайте ТОЛЬКО с официальных сайтов
- Проверяйте хеш-сумму
- Используйте антивирус

### 3. Подмена адреса

**Схема:**
- Копируете адрес для перевода
- Вредоносное ПО меняет адрес на адрес мошенника
- Деньги уходят не туда

**Защита:**
- Проверяйте первые и последние символы адреса
- Используйте QR-коды
- Сделайте тестовый перевод

### 4. SIM-swap

**Схема:**
- Мошенник получает вашу SIM-карту
- Получает SMS с кодом доступа
- Взлаывает биржевой аккаунт

**Защита:**
- Установите PIN на SIM-карту
- Используйте 2FA через приложение, не SMS
- Не связывайте номер телефона с крипто-аккаунтами

## Как купить криптовалюту безопасно?

### Безопасный путь:

1. **Зарегистрируйтесь** на бирже (Binance, Bybit)
2. **Верифицируйте** личность
3. **Включите 2FA** через приложение (не SMS!)
4. **Купите** криптовалюту
5. **Выведите** на свой кошелёк (аппаратный или мобильный)
6. **Храните** приватный ключ в безопасности

### Важно:

- НЕ храните криптовалюту на бирже
- НЕ показывайте seed-фразу никому
- НЕ переходите по ссылкам «удвоить биткоин»
- НЕ верьте «экспертам» в Telegram

## Если потеряли доступ к кошельку?

### Если есть seed-фраза:

1. Скачайте официальный кошелёк
2. Введите seed-фразу
3. Доступ восстановлен

### Если нет seed-фразы:

**Деньги потеряны навсегда.**

Нет банка, который восстановит.
Нет службы поддержки, которая поможет.

## Итог

Криптовалюта = ваши деньги = ваша ответственность.

**Правила безопасности:**

1. Используйте аппаральный кошелёк для крупных сумм
2. Запишите seed-фразу и храните офлайн
3. Никогда не вводите seed-фразу онлайн
4. Проверяйте адреса перед переводом
5. Включите 2FA через приложение
6. Не храните криптовалюту на биржах
7. Скачивайте кошельки только с официальных сайтов
