# Безопасность мобильных устройств

## Простыми словами

Смартфон — это компьютер, который всегда с вами. В нём ваши фотографии, переписки, банковские приложения, местоположение. Потеря или взлом телефона — это как потерять кошелёк, документы и дневник одновременно.

## Почему смартфоны уязвимы?

### Статистика:

- **80%** пользователей не устанавливают пароль на телефон
- **50%** не обновляют приложения регулярно
- **Число мобильных вирусов** растёт на 50% каждый год

### Риски:

- **Потеря/кража телефона** — доступ к данным
- **Вредоносные приложения** — кража данных
- **Фишинг** — кража паролей через SMS/звонки
- **Отслеживание** — шпионские приложения

## Защита телефона паролём

### Способы блокировки:

| Способ | Безопасность | Удобство |
|--------|--------------|----------|
| Графический ключ | Низкая | Среднее |
| PIN-код (4 цифры) | Средняя | Высокое |
| PIN-код (6+ цифр) | Высокая | Высокое |
| Пароль (буквы+цифры) | Очень высокая | Среднее |
| Отпечаток пальца | Высокая | Очень высокое |
| Face ID | Высокая | Очень высокое |

### Рекомендация:

**Комбинация:** Пароль/PIN + Биометрия

- Отпечаток/Face ID для быстрого доступа
- Сложный PIN/пароль как резерв

## Настройка блокировки

### Android:

1. Настройки → Безопасность → Блокировка экрана
2. Выберите метод (PIN, пароль, графический)
3. Включите «Автоблокировку» (через 30 секунд)
4. Настройте «Smart Lock» для доверенных мест

### iOS:

1. Настройки → Face ID/Touch ID и пароль
2. Введите пароль
3. Настройте Face ID/Touch ID
4. Включите «Блокировка при простое» (1 минута)

## Безопасность приложений

### Установка приложений:

✅ **Делайте:**
- Скачивайте только из Google Play / App Store
- Читайте отзывы
- Проверяйте разрешения
- Обновляйте приложения

❌ **Не делайте:**
- Не устанавливайте из неизвестных источников
- Не скачивайте «взломанные» приложения
- Не предоставляйте лишние разрешения

### Проверка разрешений:

**Опасные разрешения:**
- Доступ к камере
- Доступ к микрофону
- Доступ к местоположению
- Доступ к контактам
- Доступ к SMS

**Как проверить:**

**Android:**
Настройки → Приложения → Разрешения

**iOS:**
Настройки → Конфиденциальность

### Подозрительные признаки:

- Фонарик требует доступ к контактам
- Калькулятор требует доступ к микрофону
- Игра запрашивает доступ к SMS

## Мобильный антивирус

### Нужен ли антивирус на телефон?

**Android:** Да, рекомендуется
**iOS:** Обычно нет (песочница защищает)

### Популярные антивирусы:

- **Bitdefender** (бесплатный)
- **Kaspersky** (бесплатный)
- **Avast** (бесплатный)

### Что делает антивирус:

- Сканирует приложения
- Проверяет Wi-Fi на безопасность
- Блокирует фишинговые сайты
- Показывает разрешения приложений

## Безопасность Wi-Fi и Bluetooth

### Wi-Fi:

✅ **Делайте:**
- Отключайте Wi-Fi, когда не используете
- Не подключайтесь к открытым сетям без VPN
- Удалите сохранённые неизвестные сети

❌ **Не делайте:**
- Не включайте «Автоподключение»
- Не доверяйте сетям без пароля
- Не вводите важные данные в открытых сетях

### Bluetooth:

✅ **Делайте:**
- Отключайте Bluetooth, когда не используете
- Не принимайте соединения от незнакомцев
- Используйте режим «невидимости»

❌ **Не делайте:**
- Не оставляйте Bluetooth включённым постоянно
- Не игнорируйте запросы на сопряжение

## Бэкап данных

### Зачем нужен бэкап:

- Телефон сломается → данные не потеряются
- Телефон украдут → данные можно восстановить
- Сброс до заводских → данные вернутся

### Как сделать бэкап:

**Android:**

1. Настройки → Google → Резервное копирование
2. Включите «Резервное копирование в Google»
3. Выберите, что копировать

**iOS:**

1. Настройки → Apple ID → iCloud
2. Включите «Резервное копирование iCloud»
3. Нажмите «Создать резервную копию»

### Что копировать:

- Фотографии и видео
- Контакты
- Сообщения
- Настройки приложений
- Документы

## Найти потерянный телефон

### Android: Find My Device

1. Зайдите на google.com/android/find
2. Выберите устройство
3. Можно: позвонить, заблокировать, удалить данные

### iOS: Find My iPhone

1. Зайдите на icloud.com/find
2. Выберите устройство
3. Можно: воспроизвести звук, режим пропажи, стереть данные

### Настройте заранее!

**Android:**
Настройки → Безопасность → Найти устройство (включить)

**iOS:**
Настройки → Apple ID → Локатор → Найти iPhone (включить)

## Что делать, если телефон украли?

### Немедленно:

1. Заблокируйте через Find My Device / Find My iPhone
2. Смените пароли всех важных аккаунтов
3. Позвоните в банк (если был доступ к банковскому приложению)

### Потом:

1. Напишите заявление в полицию
2. Сообщите оператору о краже (заблокировать SIM)
3. Предупредите друзей о возможных мошеннических сообщениях

## Безопасность детей с телефонами

### Настройте родительский контроль:

**Android:**
- Family Link от Google
- Настройте время использования
- Ограничьте приложения

**iOS:**
- Экранное время
- Ограничения контента
- Спросите Siri

### Поговорите с ребёнком:

- Не публикуйте личную информацию
- Не добавляйте незнакомцев
- Не верьте «бесплатным» предложениям
- Рассказывайте, если что-то пугает

## Итог

Смартфон требует такой же защиты, как компьютер:

1. **Блокировка** — пароль + биометрия
2. **Обновления** — устанавливайте сразу
3. **Приложения** — только из официальных магазинов
4. **Бэкап** — регулярно копируйте данные
5. **Find My Device** — настройте заранее
6. **Антивирус** — для Android обязателен
