# Безопасность детей в интернете

## Простыми словами

Дети растут с интернетом как частью жизни. Они учатся, общаются, играют онлайн. Но интернет полон опасностей: незнакомцы, мошенники, насилие, неприемлемый контент. Задача родителей — научить ребёнка безопасности, а не запретить интернет.

## Какие опасности подстерегают детей?

### 1. Контент

**Что могут увидеть:**
- Насилие и жестокость
- Порнография
- Наркотики и алкоголь
- Экстремистский контент

### 2. Незнакомцы

**Риски:**
- Попытки вступить в контакт
- Онлайн-хищники
- Груминг (подготовка к встрече)
- Секстинг и шантаж

### 3. Кибербуллинг

**Что происходит:**
- Оскорбления в играх/соцсетях
- Распространение слухов
- Исключение из групп
- Угрозы насилия

### 4. Мошенники

**Схемы:**
- «Бесплатные» внутриигровые валюты
- Фишинг через игры
- Кража аккаунтов
- Покупки без разрешения

### 5. Зависимость

**Признаки:**
- Потеря интереса к реальной жизни
- Проблемы со сном
- Агрессия при отрыве от устройства
- Падение успеваемости

## Как говорить с ребёнком о безопасности?

### Правила разговора:

**❌ Не надо:**
- Кричать и пугать
- Запрещать без объяснения
- Читать переписку тайно
- Наказывать за вопросы

**✅ Надо:**
- Говорить спокойно и открыто
- Объяснять почему
- Слушать и отвечать на вопросы
- Создать доверительные отношения

### О чём говорить:

**Для маленьких (5-8 лет):**
- Не сообщай личную информацию
- Не добавляй незнакомых
- Скажи, если что-то пугает
- Не верь «бесплатному»

**Для среднего возраста (9-12 лет):**
- Пароли — только для тебя и родителей
- Не верь всему, что видишь
- Не встречайся с интернет-друзьями
- Расскажи, если обижают

**Для подростков (13-17 лет):**
- Приватность в соцсетях
- Опасности знакомств онлайн
- Цифровой след и его последствия
- Ответственность за слова

## Технические инструменты защиты

### 1. Родительский контроль

**Android (Family Link):**
1. Установите Family Link на свой и детский телефон
2. Настройте ограничения
3. Контролируйте экранное время
4. Блокируйте приложения

**iOS (Экранное время):**
1. Настройки → Экранное время
2. Ограничения контента и конфиденциальности
3. Выберите возраст ребёнка
4. Настройте время использования

**Игровые консоли:**
- **PlayStation:** Семейное управление
- **Xbox:** Xbox Family Settings app
- **Nintendo:** Родительский контроль в приложении

### 2. Безопасный поиск

**Google SafeSearch:**
1. Настройки → SafeSearch (включить)
2. Заблокировать взрослый контент

**YouTube для детей:**
1. Используйте YouTube Kids
2. Ограничьте историю просмотров
3. Отключите рекомендации

### 3. Фильтрация DNS

**Что это:**
- Блокировка взрослых сайтов на уровне сети
- Работает для всех устройств в доме

**Как настроить:**
1. На роутере или в DNS настройках
2. Используйте FamilyShield (OpenDNS):
   - 208.67.222.123
   - 208.67.220.123

### 4. Антивирус с родительским контролем

**Популярные:**
- Kaspersky Total Security
- Bitdefender Family Pack
- Norton 360 Deluxe

### 5. Контроль экранного времени

**Приложения:**
- Screen Time (iOS/Android)
- OurPact
- Qustodio

## Настройка безопасности устройств

### Компьютер/ноутбук:

1. **Учётная запись ребёнка**
   - Создайте отдельную учётную запись
   - Ограничьте права администратора

2. **Браузер**
   - Включите SafeSearch
   - Используйте детский браузер (Kiddle, Kidtopia)

3. **Приложения**
   - Разрешите только проверенные программы
   - Запретите установку новых

### Телефон ребёнка:

1. **Блокировка экрана**
   - PIN-код, который знаете только вы
   - Face ID/Touch ID

2. **Геолокация**
   - Включите, чтобы знать, где ребёнок
   - Настройте «безопасные зоны»

3. **Покупки**
   - Запретите покупки в приложениях
   - Требуйте пароль для покупок

## Правила для ребёнка

### Базовые правила:

**Конфиденциальность:**
- Никому не сообщай адрес, номер телефона, школу
- Фото только с разрешения родителей
- Используй никнейм, а не имя

**Общение:**
- Не добавляй незнакомцев
- Не верь тому, что говорят в интернете
- Не встречайся с интернет-друзьями

**Безопасность:**
- Не нажимай на рекламу
- Не скачивай без разрешения
- Не вводи пароли на незнакомых сайтах

**Если что-то плохое:**
- Расскажи родителям
- Не удаляй — это доказательство
- Сделай скриншот

### Контракт с ребёнком:

Создайте вместе «договор»:

```
Я, __________________, обещаю:
1. Не сообщать личную информацию в интернете
2. Не добавлять незнакомых людей
3. Рассказывать родителям, если что-то пугает
4. Не встречаться с интернет-друзьями без родителей
5. Использовать интернет _________________ часов в день

Родители обещают:
1. Объяснять правила, а не только запрещать
2. Не читать переписку тайно
3. Помогать, если возникнут проблемы
4. Относиться к моим увлечениям с уважением
```

## Признаки проблем

### Что должно насторожить:

**Поведенческие:**
- Ребёнок нервничает при использовании телефона
- Быстро закрывает экраны
- Не хочет рассказывать об онлайн-жизни
- Агрессия при отрыве от устройства

**Эмоциональные:**
- Депрессия или тревожность
- Проблемы со сном
- Падение успеваемости
- Потеря интереса к хобби

**Физические:**
- Головные боли от экрана
- Проблемы со зрением
- Боли в спине и шее

## Что делать, если ребёнок столкнулся с опасностью?

### 1. Не ругайте

**Ваша реакция:**
- «Спасибо, что рассказал»
- «Давай разберёмся вместе»
- «Это не твоя вина»

### 2. Соберите доказательства

- Скриншоты переписки
- Сохраните ссылки
- Не удаляйте ничего

### 3. Действуйте

**Если это кибербуллинг:**
- Заблокируйте обидчика
- Пожаловаться администрации платформы
- Сохраните доказательства

**Если это хищник:**
- Сохраните переписку
- Обратитесь в полицию
- Не удаляйте сообщения

**Если это шантаж/вымогательство:**
- Не выполняйте требования
- Сохраните доказательства
- Обратитесь в полицию (ст. 163 УК РФ)

### 4. Обратитесь за помощью

**Куда звонить:**
- **Телефон доверия для детей:** 8-800-2000-122
- **Центр «Кунгуру»:** kунгуру.рф (помощь жертвам насилия)
- **Полиция:** 102

## Итог

Безопасность детей в интернете — это баланс между защитой и доверием.

**Главные правила:**

1. **Разговаривайте** — не запрещайте
2. **Объясняйте** — почему это опасно
3. **Настройте** — техническую защиту
4. **Доверяйте** — но проверяйте
5. **Будьте примером** — сами соблюдайте правила
6. **Реагируйте спокойно** — если что-то случилось
