# План реализации: Новые типы атак и атаки между городами

## Уточнённые требования

- **Типы атак**: Только между городами (город ↔ город; города между странами)
- **Крупные страны**: 20-30 городов (США, Китай, Россия, Индия, Бразилия, Австралия)
- **Мелкие страны**: 1-2 города

#### 1.3 Обновление структуры данных атаки:
```javascript
{
    id: string,
    timestamp: string,
    sourceCity: string,      // код города (ранее sourceCountry)
    targetCity: string,      // код города (ранее targetCountry)
    sourceCountry: string,   // добавлено для отображения
    targetCountry: string,   // добавлено для отображения
    attackType: string,
    severity: string,
    sector: string,
    description: string,
    active: boolean
}
```

#### 1.4 Новые/обновлённые методы:
- `cities` - объект с базой городов
- `getCityCoordinates(cityCode)` - получить координаты города
- `generateInitialData()` - атаки только между городами
- `addNewAttack()` - случайный выбор городов из одной страны или разных
