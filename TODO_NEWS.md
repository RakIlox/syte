# Добавление просмотра подробностей новостей

## Что сделано:

### 1. Создан файл `assets/js/newsDetail.js`
- Содержит расширенную информацию о каждой новости
- Функции для открытия/закрытия модального окна
- Обработчики кликов на карточки новостей
- Поддержка закрытия по клавише Escape

### 2. Обновлён `index.html`
- Добавлен атрибут `data-news-id` для каждой карточки новости
- Добавлена разметка модального окна в конце страницы
- Подключён `newsDetail.js`

### 3. Обновлён `assets/css/news.css`
- Полные стили для модального окна новостей
- Стили для информационных блоков (warning-box, success-box, info-box)
- Адаптивность для мобильных устройств
- Анимации при открытии/закрытии

### 4. Контент новостей
Для каждой новости добавлен расширенный контент:
- Описание уязвимости/инцидента
- Технические детали
- Список подверженных систем
- Рекомендации по защите
- Ссылки на источники

## Список новостей с подробностями:
1. **cisco-vulnerability** - Critical уязвимость в Cisco ASA и FTD
2. **lazarus-linkedin** - Хакеры Lazarus атакуют через LinkedIn
3. **data-breach** - Утечка 2.7 млрд записей
4. **blacklock-ransomware** - Ransomware Blacklock
5. **browser-patches** - Патчи для Chrome и Firefox
6. **deepfake-fraud** - Мошенничество с дипфейками
7. **spyware-googleplay** - Шпионское ПО в Google Play
8. **russia-bank-ddos** - DDoS-атака на банк

## Как использовать:
1. Кликните на любую карточку новости
2. Откроется модальное окно с подробностями
3. Закройте окно кнопкой X, кликом вне окна или клавишей Escape

## Тестирование:
```bash
python3 -m http.server 8080
```
Затем откройте http://localhost:8080 и кликните на новость.
