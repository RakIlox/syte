# План мобильной оптимизации

## Проблемы, выявленные при анализе:

### 1. Стили мобильного меню
- В `main.css` есть стили для `.main-nav.active`, но в некоторых страницах (glossary.html, learn.html, about.html) они добавлены inline
- Нужно вынести в основной CSS

### 2. Тач-устройства (Touch Optimization)
- Увеличить размеры кнопок для удобства нажатия (минимум 44x44px)
- Улучшить отступы между кликабельными элементами

### 3. Размеры шрифтов
- Адаптировать размеры шрифтов для мобильных устройств (< 480px)
- Слишком крупные заголовки могут выходить за экран

### 4. Grid сетки
- `glossary.html`: grid-template-columns: repeat(4, 1fr) на мобильных может быть слишком узко
- `learn.html`: lessons-grid с 5 колонок

### 5. Modal окна
- На маленьких экранах могут обрезаться
- Нужно добавить `max-height` и `overflow-y: auto`

### 6. Карта
- На мобильных карта может занимать слишком много места
- Нужно уменьшить высоту

### 7. Скрытие контента
- Некоторые элементы могут быть скрыты или обрезаны на мобильных
