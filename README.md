# CyberThreat Explorer

Интерактивная карта кибератак в реальном времени с объяснениями для начинающих.

## Описание

Образовательный веб-проект, который визуализирует кибератаки на интерактивной карте мира и объясняет их простым языком. Цель проекта — сделать кибербезопасность понятной для всех.

## Функции

- 🌍 Интерактивная карта атак в реальном времени
- 📊 Статистика и фильтры
- 📚 Глоссарий терминов
- 🎓 Учебные материалы
- 📱 Адаптивный дизайн

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- D3.js для визуализации карты
- Font Awesome для иконок
- Google Fonts
