> Интерактивная образовательная платформа по **кибербезопасности** с визуализацией кибератак на карте мира уроками по обучению для **начинающих** и не только.

---

## 📋 Описание проекта

**CyberThreat Explorer** — образовательный веб-проект, который **визуализирует кибератаки** на интерактивной карте мира и объясняет их **простым языком**. Цель — сделать кибербезопасность **понятной** для всех.

### 🎯 Целевая аудитория
- **Начинающие** — хотят понять основы
- **Студенты** — изучают ИБ
- **Родители** — защита детей онлайн
- **Специалисты** — IT и не-IT

## 🎨 Типы атак не только те которые в легенде, а вообще все которые появляются на карте

**Полный список из интерактивной D3-карты (mapRenderer.js) — все типы атак, визуализируемые на сайте:**

| Тип | Русское название | Цвет | Краткое описание |
|-----|------------------|------|------------------|
| ddos | DDoS | #ef4444 | Распределенная атака отказа в обслуживании |
| phishing | Фишинг | #f59e0b | Социальная инженерия, поддельные сайты/письма |
| malware | Вредоносное ПО | #8b5cf6 | Вирусы, трояны, ransomware |
| scanning | Сканирование | #10b981 | Поиск уязвимостей в сети |
| bruteforce | Подбор паролей | #ec4899 | Автоматический перебор паролей |
| sqlInjection | SQL-инъекция | #06b6d4 | Внедрение SQL-кода в запросы |
| xss | XSS | #f97316 | Межсайтовый скриптинг |
| mitm | MitM | #84cc16 | Атака "человек посередине" |
| supplyChain | Цепочка поставок | #6366f1 | Компрометация через поставщиков |
| apt | APT | #0ea5e9 | Устойчивая целевая угроза |
| sessionHijacking | Перехват сессии | #f43f5e | Угон сессии пользователя |
| arpSpoofing | ARP-спуфинг | #14b8a6 | Отравление ARP-таблиц |
| cryptoAttack | Крипто | #eab308 | Атаки на криптографию |
| toctou | TOCTOU | #a855f7 | Time-of-check time-of-use |
| bufferOverflow | Переполнение | #ef4444 | Переполнение буфера |
| sslStripping | SSL Strip | #64748b | Снятие HTTPS |
| clickjacking | Clickjack | #ec4899 | Кликджекинг |
| idsEvasion | Обход IDS | #0f172a | Обход систем обнаружения |
| privilegeEscalation | Эскалация | #dc2626 | Повышение привилегий |
| logicBomb | Лог.бомба | #b91c1c | Логическая бомба |
| cloudAttack | Облако | #3b82f6 | Атаки на облачные сервисы |
| iotAttack | IoT | #8b5cf6 | Атаки на устройства IoT |
| aiAttack | AI | #06b6d4 | Атаки с ИИ |

*Примечание: Цвета соответствуют attackColors из mapRenderer.js. Легенда на карте интерактивна — кликабельна для фильтрации.*
